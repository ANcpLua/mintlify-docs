## Description

Combines a variable declaration and an immediate null-check into a single pattern match.

## Bad Code

```csharp
var x = GetValue();
if (x != null)
{
    Use(x);
}
```

## Good Code

```csharp
if (GetValue() is { } x)
{
    Use(x);
}
```

## Properties

- **Category**: Style
- **Severity**: Info
- **Enabled by default**: True
- **Code fix available**: True
