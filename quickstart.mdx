## Prerequisites

<Check>**.NET SDK 8.0+** installed</Check>
<Check>**Central Package Management** enabled in your repository</Check>

```xml title="Directory.Packages.props"
<Project>
  <PropertyGroup>
    <ManagePackageVersionsCentrally>true</ManagePackageVersionsCentrally>
  </PropertyGroup>
</Project>
```

## Installation

<Steps>
  <Step title="Add SDK reference">
    Add the SDK to your `global.json` file:

    ```json title="global.json"
    {
      "msbuild-sdks": {
        "ANcpLua.NET.Sdk": "1.3.31"
      }
    }
    ```
  </Step>
  <Step title="Update project file">
    Replace your project's `Sdk` attribute with the appropriate SDK:

    <Tabs>
      <Tab title="Library/Console">
    ```xml
    <Project Sdk="ANcpLua.NET.Sdk">
      <PropertyGroup>
        <TargetFramework>net10.0</TargetFramework>
      </PropertyGroup>
    </Project>
    ```
      </Tab>
      <Tab title="Web API">
    ```xml
    <Project Sdk="ANcpLua.NET.Sdk.Web">
      <PropertyGroup>
        <TargetFramework>net10.0</TargetFramework>
      </PropertyGroup>
    </Project>
    ```
      </Tab>
      <Tab title="Tests">
    ```xml
    <Project Sdk="ANcpLua.NET.Sdk.Test">
      <PropertyGroup>
        <TargetFramework>net10.0</TargetFramework>
      </PropertyGroup>
    </Project>
    ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Build">
    Build your project to verify the SDK is configured correctly:

    ```bash
    dotnet build
    ```
  </Step>
</Steps>

## Auto-Configured

The SDK enables:
- Nullable reference types
- LangVersion `latest`
- Deterministic builds
- ANcpLua.Analyzers + Meziantou.Analyzer
- Banned API enforcement

## Multi-Targeting

```xml
<Project Sdk="ANcpLua.NET.Sdk">
  <PropertyGroup>
    <TargetFrameworks>net10.0;netstandard2.0</TargetFrameworks>
  </PropertyGroup>
</Project>
```

Polyfills auto-injected for `netstandard2.0`/`net472`:
- Index/Range (`^1`, `..`)
- `IsExternalInit`
- String extensions

See [Polyfills](/sdk/polyfills) for details.
