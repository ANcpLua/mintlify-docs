## Quick Start

```json title="global.json"
{
  "msbuild-sdks": {
    "ANcpLua.NET.Sdk": "*"
  }
}
```

```xml title="YourProject.csproj"
<Project Sdk="ANcpLua.NET.Sdk">
  <PropertyGroup>
    <TargetFramework>net10.0</TargetFramework>
  </PropertyGroup>
</Project>
```
