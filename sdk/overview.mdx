Opinionated MSBuild SDK for .NET projects.

## Quick Start

```json title="global.json"
{
  "msbuild-sdks": {
    "ANcpLua.NET.Sdk": "1.3.31"
  }
}
```

```xml
<Project Sdk="ANcpLua.NET.Sdk"></Project>
```

## SDK Variants

| SDK | Use Case |
|-----|----------|
| `ANcpLua.NET.Sdk` | Libraries, Console, Workers |
| `ANcpLua.NET.Sdk.Web` | ASP.NET Core |
| `ANcpLua.NET.Sdk.Test` | xUnit v3 + MTP |

## Base Features

- Banned API enforcement
- [ANcpLua.Analyzers](/analyzers/overview) + Meziantou.Analyzer
- `Throw.IfNull()` guard clauses
- LangVersion `latest`, nullable enabled
- Polyfills for netstandard2.0/net472

## Web SDK Additions

- OpenTelemetry (logging, metrics, tracing)
- Health endpoints (`/health`, `/alive`)
- HTTP resilience (Polly)
- DevLogs (browser → server logs)

## Test SDK Additions

- xUnit v3 with MTP
- Coverage tooling
- FakeLogger (opt-in)

## Opt-in Features

```xml
<PropertyGroup>
  <InjectSourceGenHelpers>true</InjectSourceGenHelpers>
  <InjectFakeLogger>true</InjectFakeLogger>
  <InjectLockPolyfill>true</InjectLockPolyfill>
  <InjectTimeProviderPolyfill>true</InjectTimeProviderPolyfill>
</PropertyGroup>
```

## Opt-out

```xml
<PropertyGroup>
  <AutoRegisterServiceDefaults>false</AutoRegisterServiceDefaults>
  <GenerateClaudeMd>false</GenerateClaudeMd>
  <InjectSharedThrow>false</InjectSharedThrow>
  <IncludeDefaultBannedSymbols>false</IncludeDefaultBannedSymbols>
</PropertyGroup>
```

## DevLogs (Web SDK)

